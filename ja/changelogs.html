Change Logs

v2.1.0 Improvement

  • ヒントモードでのテキストサーチにおいて、一致する要素が1つに絞れる場合に自動的に遷移するかどうかを選択できるようになりました。
  • ブックマーク、履歴、閉じたタブをHomedicsで検索できるようになりました。
  • URLをクリップボードにコピーできる yankUrl コマンドを実装しました。
  • タブのピン留めをトグルするための togglePin コマンドを実装しました。

v2.0.1 Improvement

  • オプションページにI18nを実装して、英語と日本語に対応しました。

v2.0.0 Improvement

  • タブセレクトモードは廃止になりました。代わりにコマンドモードでタブを検索して選択できるようになりました。
  • ヒントモード中に要素内のテキストを検索してヒントを絞り込めるようにしました。候補が一致している部分はハイライトされます。
  • ヒントキーの大文字小文字を選択できるようになりました。
  • 一部コマンド名が変更になりました。
  • ブックマーク検索はブックマークフォルダも検索対象に含めるようになりました。
  • Google Chrome にインストールされているアプリもコマンドモードで検索して開けるようになりました。
  • コマンドモードの候補リストに URL が含まれていればアイコンを表示するようにしました。
  • コマンドモードでの絞り込みにおいて、候補が一致してる部分はハイライトするようになりました。
  • 全体的にデザインを改善しました。
  • その他軽微な不具合を修正しました。

v1.2.2 Improvement

  • ヒントキーの生成アルゴリズムを追加しました。
    1. Standard algorithm. 従来通りのヒントキーを生成します。
    2. Comfortable algorithm. 出来るだけ片手でヒントキーをタイプできるようにヒントキーを生成します。

v1.2.0 Improvement

  • タブ選択モードを実装しました。Google Chrome 上に開かれているタブをキーボードで選択することができます。

v1.1.1 Improvement Bugfix

  • Hometype のオプションを保存するストレージに chrome.storage を使うようになりました。それによって Hometype のオプションは各端末の Google Chrome 間で同期できるようになりました。
  • 'B' でも 'b' でも新しいタブでブックマークが開いてしまう不具合を修正しました。

v1.1.0 Improvement

  • スペースで区切ることで複数のコマンドをバインドすることができるようになりました。
  • @command と定義することで、最後に呼び出した同じコマンドのオプションを引き継いで実行することができるようになりました。
  • 各 cancelXXX コマンドは enterNormalMode に置き換わりました。

v1.0.2 Bugfix

  • ヒントキーを生成する要素をより正確に判定できるようになりました。
  • キーが押下された時にインサートモードに移行するべきかどうかを判定するようになりました。これによってインサートモードの誤検知がなくなりました。

v1.0.1 Bugfix

  • ヒントモードに area タグが含まれていなかった不具合を修正しました。

v1.0.0 Improvement

  • 継続的ヒントモードを実装しました。この機能により、ヒントモード時に要素を選んでもヒントモードから抜けずに継続して次のヒントを表示できるようになりました。これは、たとえば、複数のチェックボックスやラジオボタンを連続して選択する時に便利です。
  • Hometypeを無効にするURLが設定できるようになりました。オプションページで無効 URL が設定されていれば、それにマッチするページでは Hometype が無効になります。
  • 新しいコマンド 'noop' が実装されました。このコマンドは何もしません。キーバインドを無効にしたい時に使用してください。
  • ヒントモードでのパフォーマンスを劇的に向上しました。
  • 現在のモードを表示するのを止めました。
  • Jasmine 2.0 を使ってテストを書き、wercker を使って CI を行うようになりました。

v0.1.3 Bugfix

  • Hometype のアイコンを新しくしました。
  • restoreTab 周りの軽微な不具合を修正しました。

v0.1.2 Bugfix

  • restoreTab コマンドを実行した時にもっとも新しいタブではなく、最も古いタブを開きなおしていた不具合を修正しました。
  • Hometype がバージョンアップされた時にタブの履歴が消えていた不具合を修正しました。

v0.1.1 Improvement Bugfix

  • タブ履歴に行番号をつけました。
  • 閉じたタブの履歴を検索する時、並び順が逆になっていたのを修正しました。
  • focusFirstInput と focusLastInput の不具合を修正しました。

v0.1.0 Improvement

  • オプションページを新たに実装しました。
  • キーバインドを表示するヘルプモードを実装しました。

v0.0.3 Improvement

  • BackgroundPage の代わりに EventPage を使ってパフォーマンスを向上させました。

v0.0.2 Bugfix

  • 拡張機能の名前が Chromekey から Hometype に変更されました。
  • focusFirstInput と focusLastInput が textarea タグにフォーカスしない不具合を修正しました。

v0.0.1

  • 拡張機能が Chrome Web Storeにリリースされました。
